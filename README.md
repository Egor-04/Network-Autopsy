# Network-Autopsy

---RU---\n
Плохое интернет-соединение? Или очередная блокировка от оператора? Хотите понять из-за чего конкретную проблему и на каком уровне? Network Autopsy проводит проверку всех возможных систем.
Приложение диагностики сетевого соединения и выявления точной причины. Плохое интернет соединение, блокировки сетевых протоколов и многое другое.

--ENG---\n
Poor internet connection? Or another carrier block? Want to understand the cause of a specific problem and at what level? Network Autopsy checks all possible systems.
An app for diagnosing network connections and pinpointing the exact cause. Poor internet connection, network protocol blocking, and much more.
